## Requirements

- iOS 9.0+
- Xcode 9.0+
- Swift 4.0+


target '<Your Target Name>' do
    #pod 'AppusViper', :git => 'https://github.com/mannu2saini/VIPERLib', :tag => '1.1.0'
end
```

Then, run the following command:

```bash
$ pod install
```
